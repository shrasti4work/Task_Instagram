# Task_Instagram
Instagram mini clone
This project is a Mini Instagram Clone that allows users to:

Sign up and log in securely

Follow and unfollow other users

Create posts with image URLs and captions

Like and unlike posts

Comment on posts

View a personalized feed of posts from followed users

The application is built using modern web technologies and follows clean architecture and best practices.

Features----
Authentication

User Signup

User Login

Password hashing

Token-based authentication (JWT)


Follow System
Follow another user
Unfollow a user
Maintain followers & following relationships

Post Management
Create a post (Image URL + Caption)
View posts


Likes
Like a post
Unlike a post


Comments
Comment on posts
View commenter name and comment text


Feed
Personalized feed
Shows posts only from followed users

How to run project?
1. Start MongoDB

2. Run backend server (npm start)

3. Run frontend (npm start)

4. Open browser at http://localhost:3000
